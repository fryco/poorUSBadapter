This file documents changes in metaboard, a free AVR prototyping board design.
New entries are always appended to the end of the file. Scroll down to the
bottom to see the most recent changes.

* Release 0.0

- Changed routing so that signal tracks don't come so close to the board
  dimension.
- Allow 100 nF capacitor in 2.5 mm or 5 mm grid.
- Added mkdist.sh script so that consistent archives with PNG versions of
  board and schematics can be easily created.

* Release 1.0 2008-03-26
